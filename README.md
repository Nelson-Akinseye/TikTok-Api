# TikTok Comment Analysis Pipeline

This project is an extension of the open-source **TikTokApi** library developed by David Teather and was created as part of undergraduate research investigating discussions surrounding **dopamine detox** and digital wellbeing on TikTok. While TikTokApi provides the foundation for accessing TikTok data, this project expands its functionality into a complete research pipeline for collecting, processing, and analyzing public TikTok content.

The repository includes tools for scraping TikTok videos and comments based on user-defined search terms, extracting engagement metrics, filtering and preprocessing comment data, performing topic modeling to identify common discussion themes, and conducting sentiment analysis to measure community attitudes toward these topics. The resulting pipeline allows researchers to move from raw TikTok data to meaningful insights about how users discuss digital wellbeing interventions and respond to related content.

This project was developed for research purposes and is intended to demonstrate practical applications of web scraping, natural language processing, and machine learning techniques for large-scale social media analysis.
