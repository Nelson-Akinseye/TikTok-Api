 TIktok Scraper 

 To setup on personal computer:
 
 
 git clone <repository-url> (Click Green "code" button on git repository and copy HTTPS link) 
 cd <repository-name> 
 pip install -r requirements.txt 
 
 If requirements.txt is unavailable:

 pip install playwright
 pip install requests
 pip install httpx

 Switch to Tiktok-Api directory and run officialSearchScraper.py:
 To run officialSearchScraper.py - 
 python3 officialSearchScraper.py
